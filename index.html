Project

The main goal of this project is to create a vehicle that can move on its own without human control, ensuring safety for people. To achieve this, we used sensors like cameras, ultrasonic sensors, Bluetooth, and an Arduino module. The camera captures video to detect lane markings and traffic elements, which is processed on the Google Colab platform using fast GPU technology. The system calculates lane deviation values to decide the vehicleâ€™s direction: values between -0.99 and -0.01 mean a left turn, 0 means a straight path, and 0.01 to 0.99 mean a right turn. These values are sent to the Arduino, which controls the motors to steer the vehicle. Ultrasonic sensors are also used to detect obstacles and stop the vehicle if needed, ensuring safe movement. This project combines technology to make driving smarter and safer.
